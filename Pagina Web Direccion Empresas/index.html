Tips

Un Problema

Sólo existe un problema, normalmente no es el más evidente, sin embargo, quitando el problema solucionas todo lo demás

No es una historia

No se trata sobre una historia o consecuencias, el problema hasta una palabro un enunciado corto, directo y sencillo

No es solo dinero

No se trata solamente de problemas económico, el problema puede ser cualquier situción. Puede ser una decisión, una persona, cualquier cosa. No necesariamnete tiene que ser muy técnico o complicado.

No es el primero que encuentras

No te quedes con el primer problema que veas porque generalmente no es, analiza la situación varias veces hasta que realmente veas el problema o pule el problema hasta hacerlo un problema claro.
